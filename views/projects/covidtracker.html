ABOUT THE PROJECT

Open Source

The entire project is public on GitHub, so you can contribute ideas, tools, and code to the project

Flutter

Covid Tracker was built using Google's Flutter Framework, using the power of Dart. Flutter allows Covid Tracker to be a fast, lightweight mobile application, with a complex UI and cross platform compatibility.

Reliable

Covid Tracker uses Covid19Api, which provides data sourced from John Hopkins CSES.
