Workout Junkie


A fun, open source, workout app.

ABOUT THE PROJECT

Open Source

The entire project is public on GitHub, so you can contribute ideas, tools, and code to the project

Flutter

Workout Junkie was built using Google's Flutter Framework, using the power of Dart. Flutter allows Workout Junkie to be a fast, lightweight mobile application, with a complex UI and cross platform compatibility.

Free

Workout Junkie will be released on the Google Play Store, for free!
